# x-price-input
X-Cart Custom Pricing based on Customer Input

Version: 0.2

X-Cart: 4.6.x, 4.7.x

This module creates an option to allow custom pricing for your products in x-cart based on category id.
This module assumes:

1. Products in a category are price at zero: 0.00
2. Products have a List Price
3. Products are not variant/matrix items
